# Software para controle de estoque de tarugos e buchas

## Software criado para gerenciar o controle de estoque para facilitar no dia-a-dia no uso de peças a serem usinadas.

## Principais funções:

- Controle de entrada e saída;
- Controle de fornecedores;
- Controle de preços;
- Informações técnicas
- Perfis a serem usinados;
